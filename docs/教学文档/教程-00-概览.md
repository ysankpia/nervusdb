# 教程 00 · 全局概览

## 目标

- 了解 NervusDB 的定位、核心能力与适用场景
- 熟悉教程系列的结构、示例数据集与实践方式
- 准备后续学习所需的环境与基础知识

## 读者画像

- 有 Node.js/TypeScript 基础的开发者
- 具备图数据或知识库需求的架构师 / 数据工程师
- 准备在本地或边缘环境部署嵌入式知识库的团队

## NervusDB 一句话

> 嵌入式三元组（SPO）知识库：单文件主库 + 分页索引，支持 WAL v2、链式联想、多查询语言、属性/全文/空间索引与自动化治理。

## 示例数据集

- 默认使用 `nervusdb bench` 生成的 `demo.nervusdb` / `workshop.nervusdb`
- 结构包含：用户、项目、文件、依赖、指标等
- 教程提供 `scripts/bench-standard.mjs` 生成更大规模样例

## 环境准备

1. 安装 Node.js ≥ 18（推荐 20/22）
2. 安装 pnpm，并执行 `pnpm setup`
3. 克隆仓库并执行 `pnpm install`
4. 验证 CLI：`pnpm build && npm i -g . && nervusdb --help`

## 学习方法

- 每篇教程均包含：目标 → 概念 → 操作步骤 → 验证 → 常见问题 → 延伸阅读
- 推荐边读边运行命令，使用 `docs/使用示例` 中的脚本快速搭建环境
- 所有代码示例均为 TypeScript（ESM），可在 `tsx` 或 `vitest` 环境直接运行
