# SynapseDB 使用教程 · 概览

本教程面向初次接触与准备在生产/工程中使用 SynapseDB 的开发者，系统化介绍从安装、基本用法、查询模型与联想、事务与幂等、索引与性能优化、维护治理、存储格式、部署与最佳实践到常见问题的完整路径。你可以像学习 MySQL 一样循序渐进阅读与实践。

## SynapseDB 是什么

SynapseDB 是一个嵌入式“三元组（SPO）知识库”，以 TypeScript/Node.js 实现，强调以下特性：

- 单文件主数据 + 分页索引：`<name>.synapsedb` + `<name>.synapsedb.pages/`
- 六序索引与分页化磁盘结构：SPO/SOP/POS/PSO/OSP/OPS
- WAL v2 事务日志：BEGIN/COMMIT/ABORT、崩溃恢复与尾部安全截断
- 读快照一致性：查询链路期间固定 manifest（epoch-pin）
- 热度统计与半衰：`hotness.json` 用于热点主键分析与自动合并
- 读者注册：`readers.json` 用于维护任务在有读者时按策略跳过
- LSM-Lite 暂存（实验）：写入旁路段可按需并入索引
- 事务幂等（实验）：可选 txId/sessionId，支持跨周期去重
- 进程级写锁（可选）：同一路径单写者，读者不受影响
- CLI 工具链：检查/修复/压缩/GC/导出/热点/事务观测

## 适用场景

- 代码知识与关系图谱（文件-符号-关系）
- 轻量本地知识库/嵌入式推理
- 快速实验或边缘部署（无需独立数据库服务）
