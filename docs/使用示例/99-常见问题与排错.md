# 示例 99 · 常见问题与排错

> 配合 [教程 09 · 常见问题与排错](../教学文档/教程-09-FAQ与排错.md)，提供命令级别的实操。

## 1. CLI 报 “manifest not found”

```bash
nervusdb check demo.nervusdb --summary
nervusdb repair demo.nervusdb --fast
```

## 2. auto-compact 总是跳过

```bash
nervusdb readers demo.nervusdb
# 结束对应查询或等待再执行
```

## 3. 查询慢

```bash
nervusdb stats demo.nervusdb --summary | jq '.orders.SPO.multiPagePrimaries'
# 根据输出调整 auto-compact 或属性索引
```

## 4. WAL 爆增

```bash
nervusdb stats demo.nervusdb --summary | jq '.walBytes'
nervusdb auto-compact demo.nervusdb --auto-gc
```

## 5. 属性查询无结果

```bash
nervusdb dump demo.nervusdb POS 123 | rg '"dept"'
```

## 6. GraphQL 报错

- 检查 `docs/使用示例/graphql_usage.md`
- 使用 QueryBuilder 重现，确认语义

## 7. 幂等失效

```bash
nervusdb txids demo.nervusdb --list=10
# 确认 txId 是否重复；必要时重新生成
```

## 8. 备份恢复后异常

```bash
nervusdb check demo.nervusdb --strict
nervusdb repair demo.nervusdb --rebuild-indexes
```
