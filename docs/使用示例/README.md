## 使用方式

1. 先完成 `docs/教学文档/` 系列教程
2. 根据业务需求选择对应示例
3. 所有示例基于 `nervusdb bench` 生成的样例库，可替换为生产数据

## 注意

- 示例代码默认使用 ESM + TypeScript
- CLI 展示以 `nervusdb` 为主，亦可使用 `pnpm db:*`
- 若涉及真实数据，请先备份并在测试环境验证
