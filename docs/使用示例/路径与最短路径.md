# 变长路径与最短路径

本文展示如何使用 `followPath` 与最短路径 API。

## 变长路径（[*min..max]）

```ts
// A->B->C->D
db.addFact({ subject: 'A', predicate: 'LINK', object: 'B' });
db.addFact({ subject: 'B', predicate: 'LINK', object: 'C' });
db.addFact({ subject: 'C', predicate: 'LINK', object: 'D' });
await db.flush();

const edges = db.find({ subject: 'A' }).followPath('LINK', { min: 2, max: 3 }).all();
// 返回第2~3跳的终止边集合
```

## 最短路径（无权，BFS）

```ts
const path = db.shortestPath('A', 'D', { predicates: ['LINK'] });
```

## 最短路径（双向 BFS）

```ts
const bidir = db.shortestPathBidirectional('A', 'D', { predicates: ['LINK'] });
```

## 加权最短路径（Dijkstra）

```ts
// 权重取自边属性，如 { edgeProperties: { weight: 2 } }
const w = db.shortestPathWeighted('S', 'T', { predicate: 'R', weightProperty: 'weight' });
```

提示：变长路径默认 NODE 唯一性；可通过 `variablePath('REL', { min, max, uniqueness: 'EDGE'|'NONE' })` 调整策略。
