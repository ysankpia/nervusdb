# Legacy Platform-Era Archive

Path: `docs/archive/legacy-platform-era/`

This directory contains documentation from the pre-reset phase where NervusDB was
tracking several product identities at once: broader Cypher compatibility,
multi-language bindings, vector/search work, heavy release gates, perf windows,
and platform-style roadmaps.

Archived documents are historical evidence only. They are not current 0.1 scope,
validation policy, roadmap, or product promise.

## Inventory

- `platform-roadmap/`: old roadmaps, task lists, publishing notes, and beta
  templates.
- `cypher-tck/`: old full-Cypher, TCK, procedure, path, merge, aggregation, and
  compatibility planning.
- `bindings/`: old C/Python/Node/binding parity and ABI material.
- `vector-perf/`: old vector/HNSW, performance, beta gate, and benchmark
  material.
- `old-refactors/`: old architecture hypotheses, memos, and refactor notes.
- `misc/`: older docs that do not belong to a current harness category.

## Revival Rule

To revive archived material, write a new ADR and update product, architecture,
validation, and active plan docs. Do not link archived files from
`docs/index.md` except through this index.
