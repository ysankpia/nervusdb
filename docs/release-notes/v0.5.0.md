# Release Notes - v0.5.0 Temporal Memory Default Rollout

## Highlights

- Temporal memory（时间线存储 + 事件抽取）现已在所有 `NervusDB` 实例中默认启用。
- 提供 `db.memory.ingestMessages()` 一键对话摄取入口，与 `timelineBuilder()` 无缝协作。
- 新增系统级冒烟测试，验证新库无需配置即可写入时间记忆并查询时间线。
- README 与 `docs/benchmarks/temporal-memory.md` 更新，正式将时间记忆列为核心能力，并补充 `pnpm bench:temporal` 指南。

## Upgrade Guidance

> 当前仓库无历史用户。如从旧分支升级，仅需重新构建即可：时间记忆数据将自动存储在 `<database>.temporal.json`。若要禁用此功能，可在业务层忽略 `db.memory` API。

## Validation

- `pnpm test tests/system/temporal_memory_smoke.test.ts`
- `pnpm bench:temporal`
- `pnpm lint`
